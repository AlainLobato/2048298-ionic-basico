Atributos y etiquetas:

<a target = "_BLANK"> Abre la referencia en otra pestaña

<a href = "mailto: email"> Abre una pestaña para mandar correo al correo
puesto

<center></center> centra lo que sea que le pongamos enmedio

id = "" nos permite definir un identificador, podemos redirigir dentro
de una misma pagina con el de la forma: 

<a href = "#code"></a>
<img id = "code">

<link rel = "icon" href = "name.ico"> Esto nos permite tener una foto en
la pestaña al lado del titulo (va dentro del head)

<hr> linea que separa 

for = "NAME": atributo para asociar un label con el id de un input

<span></span> Separa para aplicar estilo de algo en especifico
<div></div> Sepata para aplicar estilo a un conjunto  


Listas:

Ordenadas: <oi>
Desordenadas: <ul>

Tienen dentro un <li> por cada elemento

value = "": atributo de li que indica desde donde iniciar la lista ordenada

style = "list-style-type: ": atributo que nos indica como se van a 
acomodar los elementos de una lista ordenada (letras, romanos, arabigo)



Multimedia:

SRC ES PARA RECURSOS Y HREF PARA HIPERVINCULOS

<img src = "LINK">
<video src= "LINK" controls = ""></video>
<audio src= "LINK" controls = ""></audio>

si le agregamos un atributo alt, podemos definir algun texto por si no
se encuentra la imagen, por otro lado, el atributo title nos permite
ver una descripcion si ponemos el cursor arriba del elemento


FORMS Y ATRIBUTOS:

<form>
	<input type = "" name = ""> 
</form>

action = "": atributo de form ruta donde se guardan los datos
method = "GET": nos crea una URL dentro de la ruta donde se guarda con los datos

method = "POST": guarda los datos y nos envia a la ruta que le dimos

DENTRO SIEMPRE HAY UN INPUT


ATRIBUTOS INPUT

placeholder = "": atributo que nos permite poner dentro de un input
algun ejemplo del valor que se dará

required = "": Para que sean forzados a llenar

<textarea> EXAMPLE </textarea>: Cuadro para mostrar o escribir info, con el
atributo readonly = "" solo se muestra, es editable en tamaño, tiene
atributo rows o cols para indicarle maximo de espacio

name = "": para darle un nombre a la "variable"

value = "": para asignarle un valor por defecto a un input (excepto textarea)

<button></button>: nos crea un boton, tiene atributo type:
- submit: envia los datos
- reset: reinicia los valores por defecto
- button: no hace nada si no se lo indicamos con JS, es mas personalizable

type:

- text: Para palabras (buscador)
- password: Para login (se oculta)
- number: Barra de numeros (no se puede escribir)
- email: nos restringe escribir un email
- color: seleccionamos color
- range: barra con un rango (con min y max como atributos podemos definir)
- date: seleccionamos fecha
- time: seleccionamos una hora
- button: boton que con el atributo value podemos modificar el texto
- sumbit: boton para enviar (igual que button pero este ya tiene nombre)
- file: para agregar un documento
- radio: "circulo" que nos permite seleccionar
- checkbox: caja que podemos seleccionar
 
METADATOS:

SEMANTICA DE HTML:

header: encabezado para presentar la pagina
nav: barra bajo el header con la que se interactua (puede estar junto al header)
article: "objetos" dentro del cuerpo de desarrollo, puede dividirse con section
aside: barra lateral
footer: pie de pagina que describe y tiene info de la pagina


TABLAS:

Se manejan con <table>, cada fila se crea con <tr> y cada columna con <td>
puede tener atributo border, el <th> se utiliza para encabezados, <thead>
agrupa la cabecera y <tbody> los elementos, <tfoot> agrega un pie de tabla


<table>
	<thead>
		<tr>
			<th>Nombre</th>
			<th>Apellido</th>
			<th>Carrera</th>
			<th>Nivel</th>
		</tr>
	</thead>
	<tbody>
		<tr>
			<td>Alain Noe</td>
			<td>Lobato Niño</td>
			<td>LCC</td>
			<td>1</td>
		</tr>
	</tbody>
	<tfoot>
		<td></td>
		<td></td>
		<td></td>
		<td>Alumnos: 1</td>
	</tfoot>
</table>
